# ☕ کافه رستوران لوتوس – پنل مدیریت منو

یک برنامه تحت وب تک‌صفحه‌ای (SPA) برای نمایش منوی کافه و یک پنل مدیریت پیشرفته جهت مدیریت آیتم‌ها، دسته‌بندی‌ها و رسانه‌ها.

## ✨ ویژگی‌ها

- 📋 **مدیریت کامل دسته‌بندی‌ها** (نام، آیکن، عکس شاخص)
- 🍽️ **مدیریت آیتم‌ها** (نام، توضیحات، قیمت، گالری تصاویر، محدودیت سفارش، افزودنی‌ها)
- 🧂 **مدیریت افزودنی‌ها** (نوشیدنی و غذا)
- 🖼️ **کتابخانه رسانه** با Infinite Scroll، کش، Lazy Loading و تولید خودکار بندانگشتی (thumbnail) با Sharp
- 🔐 **احراز هویت ادمین** با JWT
- 📱 **رابط کاربری کاملاً ریسپانسیو** (سایدبار همبرگری در موبایل)
- 🎨 **نوتیفیکیشن‌های Toast** با Notyf
- 🎯 **آیکن‌های Font Awesome** به‌صورت لوکال

## 🛠️ تکنولوژی‌ها

- **Backend:** Node.js, Express.js
- **Frontend:** HTML, CSS, JavaScript (Vanilla)
- **پردازش تصویر:** Sharp (تولید thumbnail)
- **نوتیفیکیشن:** Notyf
- **آیکن‌ها:** Font Awesome 6 (لوکال)
